# Patterns

Composed, reusable solutions.

- **Interactive proof embed:** a playable artifact inside a case study, with keyboard + pointer
  paths and an attract / playing / game-over state set.
- **Empty / loading / error states:** what shows before data, during, and on failure.
- **Reveal & motion:** scroll-reveal that settles to `transform:none`; all motion gates on
  `prefers-reduced-motion`.
- **Theme mirror:** dark default with a light architect theme via `.light` token overrides.
