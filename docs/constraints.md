# Constraints

- **Technical:** no build, no server, no backend. Single-file HTML; React + fonts via CDN;
  must work offline-after-load and deploy to any static host (GitHub Pages).
- **Brand:** dark-first defense aesthetic; monochrome base + one accent per screen; the
  connected-lighting → defense-AI narrative is the through-line.
- **Platform:** modern browsers, mobile → wide; prototypes may be embedded in host wrappers
  that carry a `transform` (don't rely on viewport-fixed positioning).
- **Legal / compliance:** WCAG 2.2 AA / AODA. No classified specifics; no fabricated metrics.
- **Timeline / resourcing:** solo designer-builder.
